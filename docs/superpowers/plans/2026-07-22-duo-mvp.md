# Duo MVP Implementation Plan

> **For agentic workers:** Execute task-by-task. Steps use checkbox syntax.

**Goal:** Ship a Vercel-native Duo MVP: 2-person rooms, WebRTC A/V + screen share, YouTube co-watch/search, audio ducking, Dinner prompts, five talk games, Mistral hybrid AI.

**Architecture:** Next.js App Router on Vercel; Supabase Realtime for WebRTC signaling/presence; P2P media + DataChannel for app state; server routes proxy YouTube Data API and Mistral.

**Tech Stack:** Next.js 15, React 19, TypeScript, Tailwind CSS 4 (or 3), Framer Motion, @supabase/supabase-js, WebRTC, YouTube IFrame API, Mistral API.

## Global Constraints

- No accounts; ephemeral rooms; max 2 peers
- Secrets only in `.env.local` / Vercel env; never `NEXT_PUBLIC_*` for Mistral/YouTube/Supabase secret
- Midnight Lounge design tokens from design spec
- Packs work without Mistral key
- Games: WYR, Word Association, Starts With, Start & End, Places — no Scribble in MVP
